my_elisp
========

My emacs lisp to customize keymap, create template file etc...

How to use
----------
1.  
    mv .emacs.d $HOME  
2.  
    mkdir -p $HOME/lib/emacs  
3.  
    cp -r elisp template $HOME/lib/emacs  
4. start emacs, customize ...
